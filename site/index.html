EvoLOCUS

GitHub Pages workbench

Blinded LOCUS evaluation in the browser.

This public site is the primary EvoLOCUS user surface. It runs entirely in your browser using aggregate LOCUS analysis artifacts, synthetic or imported bounded review queues, local browser storage for review history, and explicit exports.

Pages-first UI Aggregate-only LOCUS visuals Map + inquiry + ontology + snapshots No server database No real LOCUS rows published
Research use only. Not legal advice. This site stores reviews only in your browser unless you export them. The review queue remains synthetic unless you import a bounded queue locally. Published map and chart artifacts are aggregate-only and omit ordinance text. Real LOCUS text may contain OCR errors, model labels and scores are not legal facts, the corpus is not complete or continuously current, and users should consult official legal sources. LOCUS-v1 is licensed CC-BY-NC-4.0.

    Protocol and Provenance
    
          
    Browser-first evaluation protocol.
    
        
    
        
    
          
    
            
    Unit of evaluation
    
            
    A LOCUS chunk or synthetic demo chunk, identified by a stable source locator and record ID.
    
          
    
          
    
            
    Blinding
    
            
    Model labels and scores are hidden by default until save or explicit reveal. Reveals are logged.
    
          
    
          
    
            
    Storage
    
            
    Review events are append-only in browser localStorage. Export files are generated only on request.
    
          
    
          
    
            
    Real data
    
            
    The site does not publish real LOCUS rows. Imported queues should be bounded and license compliant.
    
          
    
          
    
            
    Score direction
    
            
    Scores are shown as neutral model scores. Directional legal interpretations remain unverified.
