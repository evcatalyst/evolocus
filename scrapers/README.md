# Scrapers

Scrapers are planned for Phase 3.

Phase 0 does not include scraper code and does not run any scraper. Future scraper work must:

- Check robots.txt and terms of service before access.
- Prefer public indexes, official bulk exports, and public PDFs.
- Use 10-15 second human-like delays by default.
- Keep maximum concurrency user-configurable.
- Log source URLs, timestamps, and retrieval outcomes.
- Use a Research Mode flag that prefers public sources and conservative access patterns.
